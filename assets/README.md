# Assets

Place your app icons here:

- `icon.png` - Main app icon (256x256 recommended)
- `icon.ico` - Windows icon (for installer)
- `tray-icon.png` - System tray icon (16x16 or 32x32)

You can generate these from a single high-resolution PNG using online tools like:
- https://icoconvert.com/
- https://convertio.co/png-ico/
